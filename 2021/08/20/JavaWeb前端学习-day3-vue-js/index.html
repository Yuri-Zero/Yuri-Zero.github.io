JavaWeb前端学习-day3-vue.js

2021年08月20日 389 字 大概 1 分钟

18号的时候学习了一部分CSS和JS语法,其实是复习了。不过由于部分原因这部分的笔记我还没整理完。
今天开始学习Vue.js,发了这篇博客做笔记
教程是看@coderwhy的《2019年coderwhy vue-vuejs从入门到精通教程》，视频本身在B站就能搜索到。配合上Vue官方的API手册进行的学习
以下为笔记的正式内容

<div id="app">
      <div>{{message}}</div>
      <div v-once>{{message}}</div>
      <!-- {{}} 是mustache语法(胡须语法)-->
      <!-- v-once标记的内容不会再被更改-->
  </div>

<script>
    //vue写法:(声明式编程) 因为是响应式所以会自动更新被修改的数据
    /*
    * 代码做了什么事情：
    * 创建vue对象的时候 传入了一些options:{}
    * {}中的el属性决定这个vue对象挂载到哪个元素上
    * {}中的data属性 这个属性用来储存一些数据
    * 这些数据可以自己定义 也可以来自于服务器
    * */
    /*
    * let 是ES6中JS新添的表示变量的关键字
    * const 是ES6中表示常量的关键字
    * let 与 var的区别
    * var只包含全局变量和函数内作用域两个作用域
    * */
    const app = new Vue({
        el:'#app',//用于挂载要管理的元素
        data:{//定义要使用的数据
            message:'Hello,World!',
            num:11
        }
    })
    /*
    原生js写法:(命令式编程)
    1.创建div元素 设置id属性
    2.定义变量
    3.显示变量
    4.修改变量
    5.重新挂载变量显示
    ↑vue只要改了就会自己变
     */
</script>
