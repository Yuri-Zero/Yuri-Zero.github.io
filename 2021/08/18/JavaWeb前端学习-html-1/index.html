JavaWeb前端学习-html-1

2021年08月18日 963 字 大概 4 分钟

今天开始学习JavaWeb相关知识，首先复习了一下简易的Html知识点，这篇博客用作笔记用。
以下为详细内容

表格、列表相关

<!--无序列表 ul 有序列表 ol-->
<ul type="none">
    <li>赵四</li>
    <li>刘能</li>
</ul>
<!-- table是表格标签
     tr是行标签
     td是单元格标签
     th是表头标签
     border 边框
     align 对齐方式
     width 宽度
     height 高度
-->
<table border="1" align="center">
    <tr>
        <th>1</th>
        <td>1</td>
        <td>1</td>
        <td>1</td>
    </tr>
    <tr>
        <td align="center">2</td>
        <td>2</td>
        <td>2</td>
        <td>2</td>
    </tr>
</table>
<!--新建一个5行5列的列表 第一行，第一列的单元格要跨两列
    colspan 跨列
    rowspan 跨行
-->
<table border="2" align="center" width="500" height="200">
    <tr>
        <td colspan="2">1</td>
        <td>1</td>
        <td>1</td>
        <td>1</td>
        <td>1</td>
    </tr>
    <tr>
        <td rowspan="2">2</td>
        <td>2</td>
        <td>2</td>
        <td>2</td>
        <td>2</td>
    </tr>
    <tr>
        <td>3</td>
        <td>3</td>
        <td>3</td>
        <td>3</td>
        <td>3</td>
    </tr>
    <tr>
        <td>4</td>
        <td>4</td>
        <td>4</td>
        <td>4</td>
        <td>4</td>
    </tr>
    <tr>
        <td>5</td>
        <td>5</td>
        <td>5</td>
        <td>5</td>
        <td>5</td>
    </tr>
</table>

小技巧 在使用IDEA时候可以使用如下代码

table>tr*5>td*5

并按tab键可以迅速生成5行5列的表格

iframe相关

<!--
iframe 标签配合a标签使用 先给iframe一个name 之后a标签target=name
width:宽
height:高
-->
<iframe src="test.html" name="myiframe" width="500" height="400"></iframe>
<a href="hello.html" target="myiframe"> 点我跳转</a>

表单、表单格式化、表单提交相关

<!--什么是表单
    表单就是html中用来收集用户信息的y用具集合-->
<!--
  form标签是表单标签
  input type ="text"是文本输入域 value=""填写默认值
  input type ="password"是密码域
  input type ="radio" 是单选 name=“”用于分组 checked="checked"为设置默认选项
  input type ="checkbox" 复选框 checked="checked"也适用
  input type = "submit" 提交
  input type = "reset" 重置
  input type = "file" 文件上传域
  input type = "hidden" 隐藏域 当我们发送信息但不需要用户看到时使用
  select 为下拉列表
      option给下拉列表提供选项 selected = "selected"为设置默认选项
  textarea为多行文本输入 rows属性设置行 cols属性设置列 起始标签和结束标签范围内的文本才是默认值
-->
<!--可以通过table来实现表单的格式化-->
<!--form action指定表单数据发送到的服务器地址 method属性设置提交的方式 默认get 还有post-->
<!--发送数据须知：
必须有name属性 否则不发送
checkbox radio 以及下拉列表的option必须设置value
表单项不在提交的form中
-->
<!-- method属性值的区别
GET特点：1.浏览器中的地址是:action属性[+?+请求参数]
请求参数格式为 name=value&name=value
如 sex=boy&hobby=C++
2.不安全【请求参数内容会直接显示到地址栏】
3.有长度限制【100个字符】
POST特点：
1.只有action属性值
2.相对比GET安全【毕竟只有action属性值】
3.理论上无长度限制
-->
<form action="http://localhost:8080" method="post">
  <h1 align="center">用户注册</h1>
  <table align="center">
    <tr>
      <td>用户名称 </td>
      <td><input type="text" value="username"/></td>
    </tr>
    <tr>
      <td>用户密码</td>
      <td><input type="password"/></td>
    </tr>
    <tr>
      <td>性别</td>
      <td><input type="radio" name="sex" checked="checked"/>男<input type="radio" name="sex"/>女</td>
      <td></td>
    </tr>
    <tr>
      <td>兴趣爱好</td>
      <td><input type="checkbox"/> Java <input type="checkbox"/> C++</td>
    </tr>
    <tr>
      <td>所在城市</td>
      <td><select>
        <option>
          ---请选择城市---
        </option>
        <option selected="selected">
          唐山
        </option>
        <option>
          天津
        </option>
      </select>
      </td>
    </tr>
    <tr>
      <td>自我评价</td>
      <td><textarea cols="10" rows="10"></textarea></td>
    </tr>
    <tr>
      <td><input type="reset"/></td>
      <td> <input type="submit"/></td>
    </tr>
  </table>
</form>

p div span相关

<!--
<div>单独占一行
<span>是行内元素衔接
<p>上行下行各空一行
-->
<div>
  这是一行文本
</div>
<span>这是文本</span>
<span>这也是文本</span>
<p>这是一行段落文本</p>
